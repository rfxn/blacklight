# blacklight

*Attackers have agents. Defenders still have grep. blacklight is the counter.*

A portable bash wrapper directed by a skills-native Managed Agent, supercharging fifteen years of Linux OSS defensive primitives — `iptables`, `mod_security`, `ClamAV`, `LMD`, `APF`, `YARA`, Apache logs, `.htaccess` walks, crontab audits — at agent speed. No platform buy-in. No fleet migration. No analyst retraining. Man-days of incident response become agentic-minutes on the substrate the defender already owns.
