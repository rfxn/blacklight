Cross-stream evidence correlator. Fuses Apache logs, ModSec events, journalctl, filesystem state, and process snapshots into one intrusion narrative.

Activate this skill whenever evidence spans two or more streams, a timeline needs reconstruction, competing hypotheses need ranking, or substrate signals appear (CVE IDs like APSB25-94, Magento paths like vendor/magento, pre-systemd markers like /sbin/init) — even if the request does not literally say "correlate."

Prefer prescribing-defensive-payloads for rule authoring; prefer extracting-iocs for IOC normalization with no cross-stream work.

Examples: "Apache 200s + ModSec deny + journalctl spike — same actor?" / "Magento PHP drop via vendor/magento" / "CentOS 6 crontab + init.d entry."
