False-positive adjudicator. Evaluates whether a ModSec deny-event, APF block, YARA/LMD signature hit, or anomaly flag is a real threat or a benign-pattern match warranting suppression.

Activate this skill whenever an alert needs an act-or-suppress decision — counter-hypothesis discipline, benign-pattern corpus lookup, scanner-threshold check, or suppression-rationale authoring — even if the request does not literally say "false positive."

Prefer prescribing-defensive-payloads for AUTHORING new rules; prefer synthesizing-evidence for cross-stream timeline reconstruction.

Examples: "ModSec rule 981172 on Magento admin POST — scanner FP or real?" / "APF block on CDN IP — suppress and allowlist" / "YARA hit on vendor/magento/framework — log suppression."
