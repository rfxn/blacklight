IOC normalizer and cluster builder. Extracts, deduplicates, and clusters IPs, domains, file hashes (MD5/SHA256), URLs, and webshell-family fingerprints from raw evidence into structured aggregation files.

Activate this skill whenever IOCs must be normalized, clustered, or fingerprinted — IPs into /24 cohorts, hashes into file-patterns.md, callback domains into actor groups, webshell strings into family signatures — even if the request does not literally say "extract IOCs."

Prefer synthesizing-evidence for timeline reconstruction or cross-stream hypothesis work; prefer gating-false-positives for alert adjudication.

Examples: "Dedup Apache access IPs, cluster by /24 subnet" / "Normalize maldet SHA256s into file-patterns.md" / "Group callback domains by registrar — same actor?"
